# 分位点清仓策略设计

## 背景

现行策略（恒定市值法）使用固定偏离度阈值（MA250 偏离 +50%/+35%/+30%）触发清仓，
用偏离度 < +3% 解除冷却期。固定阈值对不同市场的"极值"定义不一致，
且不能反映当前价格在历史中的真实位置。

## 目标

在保持恒定市值法买卖逻辑不变的前提下，用价格历史分位点替代偏离度阈值，
使清仓和解冻条件更贴近各市场的真实高低位。

## 核心设计

### 分位点计算

**扩展窗口（expanding window）**：在每个定投日 T，计算：

```
pct_rank = count(price_history[0..T] <= price_T) / count(price_history[0..T])
```

- 使用数据起点至当日所有历史价格
- 无前瞻偏差，实盘可直接复用
- 各 ETF 独立计算，不跨标的

### 触发条件替换

| 条件 | 原逻辑 | 新逻辑 |
|---|---|---|
| 清仓触发 | 偏离度 ≥ 阈值（+50%/+35%/+30%） | 价格分位点 ≥ `liquidate_pct` |
| 冷却解除 | 偏离度 < +3% | 价格分位点 ≤ `resume_pct` |

其余逻辑（目标市值冻结、期数 offset、恒定市值买卖差额计算）完全不变。

### 参数统一

各标的使用同一套 `(liquidate_pct, resume_pct)` 参数，
分位点本身已体现各市场特性，无需按标的单独设定。

## 网格搜索

- 清仓分位点候选：80%、85%、90%、92%、95%
- 再入分位点候选：30%、40%、50%、60%、70%
- 共 25 组组合，要求 `resume_pct < liquidate_pct`
- 优化目标：20 年回测年化收益率
- 同时输出 2 年、5 年、10 年周期结果供对比
