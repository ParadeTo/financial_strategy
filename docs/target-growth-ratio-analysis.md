# 增量阶段目标增速比例分析

## 背景

策略分两个阶段：
- **第一阶段（累计投入 < 15万）**：常规定投，目标市值每期增长 BASE_AMOUNTS（沪深300: 1800 / 上证50: 1200 / 恒生: 1500 / 标普: 1500，合计 6000 元/期）
- **第二阶段（累计投入 ≥ 15万）**：暂停常规定投，每期 2000 元增量资金进入储备金池，仅保留网格收割和大跌加码

第二阶段的目标市值仍需持续增长，否则持仓容易超过目标，频繁触发收割导致卖早。

**核心问题：第二阶段目标增速设为第一阶段的多少比例？**

## 测试方案

### 曾尝试的动态增速（已放弃）

```python
growth = min(base, max(increment_base, reserve_pool * weight / 10))
```

问题：储备金花光后增速骤降 → 目标偏低 → 过早收割 → 回收资金又不够 → 恶性循环。

### 固定增速方案

```python
INCREMENT_BASES = [b * TARGET_GROWTH_RATIO for b in BASE_AMOUNTS]
```

测试了三个比例：33%、50%、70%。

## 回测结果对比

基于 10 个窗口（4×2年、3×5年、2×10年、1×20年）的回测数据：

| 窗口 | 33% | 50% | 70% |
|------|-----|-----|-----|
| 2年A(24-26) | +20.9% | +21.0% | +21.1% |
| 2年B(22-24) | +2.8% | +3.0% | +3.3% |
| 2年C(20-22) | -2.5% | -4.6% | -4.4% |
| 2年D(18-20) | +1.9% | +1.4% | +1.1% |
| 5年A(21-26) | +26.5% | +29.2% | +28.4% |
| 5年B(19-24) | +3.3% | +2.9% | +4.7% |
| 5年C(17-22) | +12.3% | +9.8% | +11.1% |
| 10年A(16-26) | +50.0% | +54.4% | +56.5% |

| **20年(06-26)** | **+64.6%** | **+80.2%** | **+86.0%** |

年化收益率：

| 窗口 | 33% | 50% | 70% |
|------|-----|-----|-----|
| 2年均值 | 2.6% | 2.6% | 2.6% |
| 5年均值 | 2.8% | 2.7% | 3.0% |
| 10年均值 | 3.9% | 4.1% | 4.3% |
| 20年 | 4.2% | 5.0% | 5.3% |

## 分析

- **短周期（2年）**：三个比例差异极小，因为大部分窗口还没进入增量阶段，或刚进入不久
- **中周期（5年）**：70% 略优，尤其在 5年B(19-24) 窗口优势明显（+4.7% vs +2.9%/+3.3%）
- **长周期（10年、20年）**：70% 显著优于 33% 和 50%，20年收益率从 64.6% 提升到 86.0%

**70% 更优的原因**：目标增速更快 → 目标市值更高 → 牛市中不会过早触发收割 → 持仓能充分享受上涨。同时 70% 不像 100%（即与第一阶段相同）那样激进，仍然留出了足够的收割空间。

## 结论

**采用 70% 作为增量阶段的目标增速比例。**
